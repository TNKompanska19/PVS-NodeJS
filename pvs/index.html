DHT11 Sensor

The DHT11 sensor is a popular sensor for measuring temperature and humidity. It provides a digital signal to the Arduino, making it easy to integrate into various projects.

BH1750 Sensor

The BH1750 sensor is a digital light sensor that measures ambient light intensity. It communicates with the Arduino using the I2C protocol and is commonly used in projects that require light level measurements.
